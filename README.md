# 🎮 CLICKY

> **Click. Compete. Conquer.**

CLICKY is a modern **real-time multiplayer click battle game** built using **HTML, CSS, JavaScript, and Socket.IO**.  
Players can create private rooms, invite friends, and compete in high-speed clicking battles with live updates, countdowns, milestones, and animated results.

---

# ✨ Features

## 🏠 Room System
- Create private multiplayer rooms
- Join rooms using unique room codes
- Supports **2–3 players**
- Real-time lobby updates

## ⚡ Real-Time Gameplay
- Live click synchronization using Socket.IO
- Animated click counters
- CPS (Clicks Per Second) tracker
- Live progress bars

## 🎯 Competitive Mechanics
- Countdown before match starts
- Multiple round durations
- Real-time ranking system
- Winner podium screen

## 🔥 Milestone System
Unlock milestone effects while clicking:
- 🔥 ON FIRE
- ⚡ LIGHTNING
- 💥 BOOM
- 🚀 ROCKET
- 👑 ROYALTY
- 🌈 GODLIKE

## 🛡️ Host Controls
- Kick players
- Ban players
- Unban system
- Round timer controls

## 🎨 Modern UI/UX
- Fully responsive design
- Neon dark theme
- Animated buttons & ripples
- Smooth transitions
- Confetti celebration effects
